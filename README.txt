# dice-quiz


A small implementation to solve a specific dice quiz based on an online article in the German Zeit Newspaper.



# Language

General descriptions, file names and commit languages will be in English.

Analysis and solution-based content will be in German language.

The implementation itself will be in English.



# User Manual

1. Install and configure a webbrowser that is capable of JavaScript.

2. Open the folder "Quiz Analysis and Prototype" and read the file "Quiz - Introduction - Analysis - Solutions.txt".

3. In the same folder open the file "dice-quiz-proto.html" with your webbrowser.
